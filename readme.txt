Run the following commands to start the front and server:

Front:
1)write npm install in front folder in terminal.
2)Then run npm start
3)It will run on localhost:3000

Server:
1)Write npm install in server folder in terminal.
2)Then run npm start
3)Server will run on localhost:4200

For CRUD operation you can use POSTMAN

Endpoints of the APP:

1)GET:  
localhost:4200
2)POST:
localhost:4200/add
And provide the data in body.
3)Delete:
localhost:4200/id
Here id is the unique code of the user you want to delete
4)UPDATE:
localhost:4200/update/id
Here id is the unique code of the user you want to update
